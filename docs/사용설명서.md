## 1. 소개

**L-CAM Doctor**는 Huvitz 덴탈 밀링머신 사용자를 위한 오프라인 문제 해결 도우미입니다.

### 주요 특징

- **오프라인 사용 가능**: 인터넷 연결 없이도 에러 검색 및 해결 가이드 확인 가능
- **자동 로그 분석**: 장비 로그 파일을 분석하여 최근 발생한 에러를 자동으로 감지
- **단계별 해결 가이드**: 각 에러에 대한 상세한 해결 방법 제공
- **간편한 리포트 전송**: 해결되지 않는 문제는 로그 파일과 함께 기술 지원팀에 전송

| 메모리 | 4GB RAM |

## 3. 설치 방법

### 3.1 설치 파일 실행

1. 제공받은 설치 파일(`L-CAM Doctor Setup.exe`)을 더블클릭합니다.
2. 화면의 안내에 따라 설치를 진행합니다.
3. 설치가 완료되면 바탕화면에 **L-CAM Doctor** 아이콘이 생성됩니다.

### 3.2 포터블 버전 사용

설치 없이 사용하려면:
1. 제공받은 ZIP 파일의 압축을 해제합니다.
2. 폴더 내의 `L-CAM Doctor.exe` 파일을 실행합니다.

---

## 4. 프로그램 실행

바탕화면의 **L-CAM Doctor** 아이콘을 더블클릭하여 프로그램을 실행합니다.

프로그램이 시작되면 메인 화면이 표시됩니다.

---

## 5. 주요 기능

### 5.1 에러 검색

장비에 표시된 에러 코드나 증상을 검색하여 해결 방법을 찾을 수 있습니다.

#### 사용 방법

1. 메인 화면 상단의 **검색창**에 에러 코드 또는 증상을 입력합니다.
   - 예시: `204`, `과부하`, `온도`, `센서` 등
2. 입력하면 즉시 관련 에러 목록이 표시됩니다.
3. 원하는 에러를 클릭하면 상세 해결 가이드로 이동합니다.

#### 검색 팁

- **에러 코드로 검색**: 장비 화면에 표시된 숫자를 그대로 입력
- **증상으로 검색**: "모터", "온도", "센서", "통신" 등 키워드 입력
- **보드명으로 검색**: "MAIN", "SPIN", "ATC" 등 보드 이름 입력

---

### 5.2 로그 분석

장비에서 추출한 로그 파일을 분석하여 최근 발생한 에러를 자동으로 감지합니다.

#### 자동 감지 (권장)

L-CAM에서 로그를 추출하면 자동으로 감지됩니다:

1. 메인 화면의 **"최근 장비 로그"** 섹션을 확인합니다.
2. 감지된 로그 파일 옆의 **[분석]** 버튼을 클릭합니다.
3. 분석이 완료되면 최근 발생한 에러 목록이 표시됩니다.

#### 수동 업로드

로그 파일을 직접 선택하려면:

1. **"파일을 드래그하거나"** 영역에 로그 파일을 끌어다 놓습니다.
2. 또는 **[파일 선택]** 버튼을 클릭하여 파일을 선택합니다.

> **참고**: 분석 가능한 파일은 `.2.txt`로 끝나는 에러 로그 파일입니다.
> 예: `log_10.82.79.2.txt`

#### 분석 결과 화면

분석이 완료되면 다음 정보가 표시됩니다:

- **총 에러 수**: 분석 기간 동안 발생한 총 에러 횟수
- **고유 에러 수**: 서로 다른 에러 종류의 개수
- **에러 목록**: 최신순으로 정렬된 에러 목록
  - 발생 시각
  - 심각도 (CRITICAL / WARNING / INFO)
  - 에러 코드 및 설명
  - **[해결 방법 보기]** 버튼

---

### 5.3 해결 가이드

각 에러에 대한 단계별 해결 방법을 제공합니다.

#### 해결 가이드 화면 구성

1. **에러 정보**: 에러 코드, 제목, 심각도, 상세 설명
2. **해결 단계**: 순서대로 따라할 수 있는 단계별 가이드
   - 각 단계에는 구체적인 조치 방법이 설명되어 있습니다
   - 필요한 경우 참고 이미지나 추가 설명이 포함됩니다
3. **액션 버튼**:
   - **[해결됨]**: 문제가 해결되면 클릭
   - **[해결 안 됨 (로그 보내기)]**: 해결되지 않으면 클릭하여 기술 지원 요청

---

### 5.4 로그 리포트 전송

해결 가이드를 따라해도 문제가 해결되지 않는 경우, 로그 파일을 기술 지원팀에 전송할 수 있습니다.

#### 리포트 전송 방법

1. 해결 가이드 화면에서 **[해결 안 됨 (로그 보내기)]** 버튼을 클릭합니다.
2. 고객 정보를 입력합니다:
   - **성함**: 담당자 이름
   - **업체명**: 치과/병원 이름
   - **연락처**: 연락 가능한 전화번호
3. 로그 파일을 첨부합니다:
   - 드래그 앤 드롭 또는 파일 선택
4. **[리포트 생성 및 저장]** 버튼을 클릭합니다.

#### 전송 결과

- 리포트 ZIP 파일이 **바탕화면**에 저장됩니다.
- 동시에 기술 지원팀 이메일로 자동 전송됩니다.
- 기술 지원팀에서 분석 후 연락드립니다.

## 6. 로그 파일 추출 방법

L-CAM 프로그램에서 장비 로그를 추출하는 방법입니다.

### 단계별 안내

1. **L-CAM** 프로그램을 실행합니다.
2. **[EXPORT]** 단계로 이동합니다.
3. 장비 리스트에서 본인의 **장비 이름**을 선택합니다.
4. 마우스 **오른쪽 버튼(Right Click)**을 클릭합니다.
5. 메뉴에서 **'Extract Machine Logs'**를 선택합니다.
6. 로그 추출이 완료될 때까지 기다립니다.
7. L-CAM Doctor의 **[새로고침]** 버튼을 클릭하여 로그를 감지합니다.

### 로그 파일 저장 위치

추출된 로그 파일은 다음 경로에 저장됩니다:

```
C:\Huvitz\Lilivis\L-CAM\log\machine_logs\Lilivis Mill
```

## 7. 자주 묻는 질문 (FAQ)

### Q1. 인터넷 연결 없이도 사용할 수 있나요?

**A:** 네, L-CAM Doctor는 오프라인에서도 에러 검색과 해결 가이드 확인이 가능합니다. 단, 로그 리포트 이메일 전송 기능은 인터넷 연결이 필요합니다.

### Q2. 로그 파일이 감지되지 않습니다.

**A:** 다음 사항을 확인해주세요:
- L-CAM에서 로그를 정상적으로 추출했는지 확인
- [새로고침] 버튼을 클릭
- 로그 파일이 올바른 경로에 있는지 확인

### Q3. 분석 결과에 "에러가 없습니다"라고 표시됩니다.

**A:** 기본 설정으로 최근 7일간의 로그만 분석됩니다. 이 기간 동안 에러가 발생하지 않았다면 정상적인 결과입니다.

### Q4. 에러 코드를 검색해도 결과가 없습니다.

**A:** 해당 에러가 데이터베이스에 등록되지 않은 경우입니다. [해결 안 됨] 버튼을 통해 로그 파일을 전송해주시면 기술 지원팀에서 분석 후 안내해드립니다.

### Q5. 프로그램이 실행되지 않습니다.

**A:** 다음을 확인해주세요:
- Windows 10 이상인지 확인
- 바이러스 백신이 프로그램을 차단하지 않았는지 확인
- 관리자 권한으로 실행 시도

## 8. 문제 해결

### 프로그램이 느리게 동작하는 경우

- 큰 로그 파일(수백 MB)을 분석할 때 시간이 걸릴 수 있습니다.
- 분석이 완료될 때까지 잠시 기다려주세요.

### 화면이 제대로 표시되지 않는 경우

- 프로그램을 종료한 후 다시 실행해보세요.
- 모니터 해상도가 1024x768 이상인지 확인하세요.

### 리포트 전송이 실패하는 경우

- 인터넷 연결 상태를 확인하세요.
- 회사 방화벽이 전송을 차단할 수 있습니다.
- 전송이 실패해도 ZIP 파일은 바탕화면에 저장되므로, 이메일로 수동 전송 가능합니다.
